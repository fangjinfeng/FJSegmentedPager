# FJSegmentedPager


图片选择器:一句话集成系统相册、拍照、手机相册等选取图片。

# 集成方法

1. 静态：手动将``FJSegmentedPager``文件夹拖入到工程中。
2. 动态：``CocoaPods：pod 'FJSegmentedPager', '~> 1.0.0``。
